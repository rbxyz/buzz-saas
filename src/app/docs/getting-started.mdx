# Início Rápido

Este guia irá te orientar no processo de configuração do ambiente de desenvolvimento local para o `buzz-saas`.

## Pré-requisitos

Antes de começar, certifique-se de que você tem o seguinte software instalado em sua máquina:
- **Node.js**: Versão 20.x ou superior.
- **npm**: Gerenciador de pacotes do Node.js.
- **Git**: Para controle de versão.
- **Docker**: Para executar uma instância local do PostgreSQL.

## 1. Clonar o Repositório

Primeiro, clone o repositório do projeto do GitHub para a sua máquina local:

```bash
git clone https://github.com/rbxyz/buzz-saas.git
cd buzz-saas
```

## 2. Instalar Dependências

Instale todas as dependências do projeto usando npm:

```bash
npm install
```

## 3. Configurar o Banco de Dados

O projeto utiliza PostgreSQL. A maneira mais fácil de executá-lo localmente é usando Docker.

1.  Crie e execute um container do PostgreSQL (se ainda não tiver um):
    ```bash
    docker run --name buzz-saas-db -e POSTGRES_USER=docker -e POSTGRES_PASSWORD=docker -e POSTGRES_DB=buzz-saas -p 5432:5432 -d postgres
    ```
2.  Isto irá criar um banco chamado `buzz-saas` com usuário e senha `docker`.

## 4. Configurar Variáveis de Ambiente

Crie um arquivo chamado `.env` na raiz do projeto. Preencha seu arquivo `.env` com os seguintes valores, no mínimo:

```env
# URL de conexão com o banco de dados que você acabou de criar.
# Formato: postgresql://<user>:<password>@<host>:<port>/<database>
DATABASE_URL="postgresql://docker:docker@localhost:5432/buzz-saas"

# Chave secreta para autenticação JWT.
# Você pode usar 'openssl rand -hex 32' no terminal para gerar uma chave segura.
AUTH_SECRET="sua_chave_secreta_aqui"
```

## 5. Aplicar as Migrações do Banco

Com o banco de dados em execução e o `.env` configurado, aplique o schema mais recente ao banco:

```bash
npm run db:push
```
Isso criará todas as tabelas necessárias no seu banco de dados local.

## 6. Executar a Aplicação

Finalmente, inicie o servidor de desenvolvimento:

```bash
npm run dev
```

A aplicação estará disponível em [http://localhost:3000](http://localhost:3000).
